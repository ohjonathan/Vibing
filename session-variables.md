# Session Variables

> **Note:** For each feature, copy `session-template.md` and fill variables there.
> This file is a reference for variable definitions.

---

## Variable Reference

| Variable | Description | Example |
|----------|-------------|---------|
| `PROJECT` | Project name | `Project Ontos` |
| `VERSION` | Current version being built | `v2.9.4` |
| `PREV_VERSION` | Previous version | `v2.9.3` |
| `BRANCH` | Git branch name | `feature/config-merge` |
| `SPEC_PATH` | Path to current implementation spec | `docs/v2.9.4-spec.md` |
| `PR_URL` | Pull request URL (fill after PR created) | `https://github.com/user/repo/pull/42` |

---

## Usage

1. Copy `session-template.md` to `sessions/[version].md`
2. Fill in variables at the top of that file
3. The template provides ready-to-copy prompts with placeholders
